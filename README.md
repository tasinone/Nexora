## Installation

**Nexora** can be manually installed in your browser by following these steps:

### **1Ô∏è‚É£ Download the Extension**

1. Go to the Releases page.
2. Download the latest `.zip` release file.
3. Extract/unzip the downloaded file to a folder on your computer.

### **2Ô∏è‚É£ Install in Google Chrome**

1. Open Chrome and go to: `chrome://extensions/`
2. Enable **Developer mode** (toggle switch at the top right).
3. Click **Load unpacked**.
4. Navigate to the folder where you unzipped Nexora and select it.
5. The extension will now appear in your Chrome extensions list.

### **3Ô∏è‚É£ Install in Mozilla Firefox**

1. Open Firefox and go to: `about:debugging#/runtime/this-firefox`
2. Click **Load Temporary Add-on**.
3. Navigate to the folder where you unzipped Nexora.
4. Select any file inside the folder (Firefox will load the extension).
5. The extension will now be active in your Firefox browser.

> ‚ö†Ô∏è Note for Firefox: This temporary install will disappear when you restart Firefox. For permanent usage install from Firefox Add-ons.

### **4Ô∏è‚É£ Customize Your Startpage**
